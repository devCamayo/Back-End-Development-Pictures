# Back-End-Development-Pictures

## 📸 Servicio de Imágenes - Backend

Este servicio proporciona una API RESTful para la gestión de imágenes, desarrollada con Flask. Permite crear, leer, actualizar y eliminar imágenes a través de endpoints REST. Es parte del proyecto final del curso IBM Back-End Development.
